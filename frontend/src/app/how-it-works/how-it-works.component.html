# Введение

Главной используемой технологией для выполнения прикладной вышеобозначенной задачи будет язык программирования Python. В настоящее время Python является одним из наиболее популярных языков программирования [1]. Популярность его обеспечена в том числе и наличеем большого количества качественно спроектированных, хорошо документированных библиотек для решения прикладных задач.
